This directory contains example Topographica simulation scripts,
including the following simple ones:

  tiny.ty              - Short, simple, fast example
  hierarchical.ty      - Example of connecting multiple maps
  gcal.ty              - Orientation map with gain control and homesotatic
                         plasticity
  perrinet_retina.ty   - Example of wrapping objects in an external simulator
                         as Topographica Sheets
  perrinet_retina_pynest.py
  som_retinotopy.ty    - SOM retinotopy map
  ptztracker.ty        - An example of using a pan/tilt/zoom camera to do
                         simple visual tracking
  saccade_demo.ty      - Saccade demo
  lissom_audio.ty      - Example of a simplified auditory pathway incorporating
                         the cochlea, MGBv, and AI
